motivewave_pivots_formulas

Question

根据 MotiveWave 官方指标源码 PivotPoints.java，解释并列出 Classic、Standard、Woodie、Camarilla、Fibonacci 等不同类型 Pivot Points 的数学计算公式。

Answer

根据 MotiveWave 官方指标源码 PivotPoints.java，各类型枢轴线的基础变量和计算公式如下：

基础定义数据

  • H = 前一日最高价 (High)
  • L = 前一日最低价 (Low)
  • C = 前一日收盘价 (Close)
  • TO = 前一日开盘价 (Open)
  • range = H - L

1. Classic Pivot (经典枢轴)

  • 中枢轴 (P) = (H + L + C) / 3
  • 阻力位 (Resistance):
    • R1 = (P * 2) - L
    • R2 = P - S1 + R1
    • R3 = 2 * (P - L) + H
  • 支撑位 (Support):
    • S1 = (P * 2) - H
    • S2 = P - (R1 - S1)
    • S3 = L - (2 * (H - P))

2. Standard Pivot (标准枢轴)

  • 中枢轴 (P) = (H + L + C) / 3
  • 阻力位 (Resistance):
    • R1 = (P * 2) - L
    • R2 = P + range
  • 支撑位 (Support):
    • S1 = (P * 2) - H
    • S2 = P - range

3. Woodie Pivot (伍迪枢轴)

中轴的计算引入了前一日的开盘价，以对近期价格赋予更高权重。
* 中枢轴 (P) = (H + L + C + TO) / 4
* 阻力位 (Resistance):
* R1 = (2 * P) - L
* R2 = P + range
* R3 = H + 2 * (P - L)
* R4 = R3 + range
* 支撑位 (Support):
* S1 = (2 * P) - H
* S2 = P - range
* S3 = L - 2 * (H - P)
* S4 = S3 - range

4. Camarilla Pivot (卡玛里拉枢轴)

以昨日收盘价 C 直接作为 Central Pivot 枢轴线，阻力与支撑系数采用 1.1 的固定倍数缩放。
* 中枢轴 (P) = C
* 阻力位 (Resistance):
* R1 = C + range * 1.1 / 12
* R2 = C + range * 1.1 / 6
* R3 = C + range * 1.1 / 4
* R4 = C + range * 1.1 / 2
* 支撑位 (Support):
* S1 = C - range * 1.1 / 12
* S2 = C - range * 1.1 / 6
* S3 = C - range * 1.1 / 4
* S4 = C - range * 1.1 / 2

5. Fibonacci Pivot (斐波那契枢轴)

  • 中枢轴 (P) = (H + L + C) / 3
  • 阻力位 (Resistance):
    • R1 = P + 0.382 * range
    • R2 = P + 0.618 * range
    • R3 = P + 1.000 * range
    • R4 = P + 1.618 * range
  • 支撑位 (Support):
    • S1 = P - 0.382 * range
    • S2 = P - 0.618 * range
    • S3 = P - 1.000 * range
    • S4 = P - 1.618 * range

6. Fib Zone (斐波那契区域 1)

主要涵盖 0.5、0.618、1.0 和 1.382 扩展倍数，并在指定的阻力/支撑阶梯中建立背景填充区域（Zone）。
* 中枢轴 (P) = (H + L + C) / 3
* 阻力位 (Resistance):
* R1 = P + 0.5 * range, R2 = P + 0.618 * range, R3 = P + range, R4 = P + 1.382 * range
* 支撑位 (Support):
* S1 = P - 0.5 * range, S2 = P - 0.618 * range, S3 = P - range, S4 = P - 1.382 * range
* 填充区域 (Zone): [R3, R4]、[R1, R2]、[S1, S2]、[S3, S4]

7. Fib Zone 2 (斐波那契区域 2)

* R7 = P + 1.618 * range, R8 = P + 2.618 * range
* 支撑位 (Support):
